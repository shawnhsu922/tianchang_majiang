# 一、产品概述（Product Overview）

## 产品目标

开发一款基于微信小程序的四人实时联网麻将游戏，完整还原天长麻将线下玩法。

产品第一阶段不考虑商业化，不考虑 AI，不考虑排行榜。

目标只有一个：

> 四位玩家能够像在线下棋牌室一样，稳定、流畅、公平地完成一局麻将。

---

## 产品定位

棋牌室线上替代品。

适用于：

- 熟人组局
- 微信好友
- 家人朋友

强调：

- 操作简单
- 节奏快
- 规则准确
- 网络稳定

# 六、产品原则（Product Principles）

1. 游戏规则必须100%还原线下玩法。

2. 所有计算必须由服务器完成。

3. 所有玩家必须保持实时同步。

4. 所有花数必须能够展示完整计算过程。

5. 玩家不得修改规则。

6. 第一版本稳定性高于功能数量。

---

# 七、验收标准（Acceptance Criteria）

V1.0 满足以下条件即可上线测试：

- 四名玩家能够正常创建房间。

- 四名玩家能够完成完整一局麻将。

- 所有规则判定正确。

- 花数计算正确。

- 掉线能够恢复。

- 托管能够正常运行。

- 结算金额正确。

- 游戏过程中不存在不同步问题。
