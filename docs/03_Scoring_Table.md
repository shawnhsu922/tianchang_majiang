# 二、计分原则

最终花数采用累加制。

计算公式：

最终花数 = 基础胡型 + 特殊牌型 + 事件奖励 + 附加奖励

其中：

- 基础胡型只能选择一种。
- 特殊牌型允许叠加（规则特别说明除外）。
- 事件奖励允许叠加。
- 附加奖励允许叠加。
